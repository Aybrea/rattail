# 快速开始

## 总览

`Rattail` 是一个面向前端开发人员的实用工具库，轻量级且 `ts` 友好

### 安装

::: code-group

```shell [npm]
npm i rattail -S
```

```shell [yarn]
yarn add rattail
```

```shell [pnpm]
pnpm add rattail
```

:::

### 使用

```ts
import { isString } from 'rattail'

console.log(isString('rattail'))
```

### 为什么不是 Lodash？

`Rattail` 不是 [Lodash](https://lodash.com/) 的替代品，也不是函数式编程解决方案。`Rattail` 旨在为更多应用场景提供更加实用的工具。
