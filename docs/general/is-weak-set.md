# isWeakSet

Determine whether the input value is a `WeakSet`.

### Usage

```ts
import { WeakSet } from 'rattail'

isWeakMap(new WeakSet()) // return true
```

### Arguments

| Arg     | Type  | Defaults |
| ------- | :---: | -------: |
| `value` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
