# isError

Determine whether the input value is an `Error` object.

### Usage

```ts
import { isError } from 'rattail'

isError(new Error('message')) // return true
```

### Arguments

| Arg   | Type  | Defaults |
| ----- | :---: | -------: |
| `val` | `any` |          |

### Return

|   Type    |
| :-------: |
| `boolean` |
